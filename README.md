# 皇家鋁材有限公司 (Royal Aluminum)

**「從手作溫暖，到質感生活。」**

本專案/網站隸屬於母公司 **皇家鋁材有限公司**（網域註冊為 `royaltech.com.tw`）。我們發跡於台灣台南，致力於將精湛的傳統工藝與現代生活痛點結合，提供消費者兼具美感與實用性的質感選品。

## 旗下品牌與核心產品

### 🔸 逸沐選品 (Yimu Curation)
我們最新推出的質感生活提案品牌，專注於解決日常與旅行中的收納痛點。
*   **主力商品：FoldGo 可折疊大容量旅行包**
    *   **設計理念**：捨棄笨重空箱，打造能「隨時變身」的聰明裝備。
    *   **核心機能**：去程可摺疊成「公事包」大小輕巧起步；回程展開釋放 66 公分超大容量(約當24吋行李箱)，且底部內建「耐磨滾輪」，裝滿戰利品也能傾斜優雅拖著走，徹底取代傳統笨重行李箱。

### 🔸 Royal Tech 皇家工藝
延續我們創立的初衷，專注於台灣在地手工皮革設計。
*   **核心商品**：精品客製化皮件、Q 版神明祈願吊飾（財神爺、關聖帝君等），將傳統信仰文化轉化為有溫度的隨身配件。

## 網頁專案技術說明 (Landing Page)
針對「逸沐 FoldGo 旅行包」的單頁式（One-Page）導購網頁，我們採用了以下技術與架構：
*   **技術棧**：純 HTML5、CSS3、Vanilla JavaScript。
*   **視覺風格**：結合質感選物店的極簡留白（瑪黑風），與高轉換率的在地化募資文案。
*   **核心功能**：
    *   全站響應式設計（RWD），完美適配手機與桌機瀏覽。
    *   自訂 `Before & After` 互動式變身滑塊（無需依賴外部套件）。
    *   Intersection Observer API 實作平滑的滾動浮現（Scroll Reveal）動畫。
    *   完善的 SEO 語意化標籤與長尾關鍵字佈局。
